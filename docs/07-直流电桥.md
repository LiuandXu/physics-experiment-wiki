# 直流电桥

> **说明**：本笔记依据《大学物理实验》“实验7 直流电桥实验”整理。该实验涵盖直流单臂电桥（惠斯通电桥）、双臂电桥（开尔文电桥）、非平衡电桥及热敏电阻测量四部分。原文为扫描图片版 PDF，共 7 页（第 7 页末尾混入“实验8 电子束”开头，未纳入本笔记）。公式由原文图示与上下文重建，电路图以占位标记标注待提取/重绘。

## 实验目的

1. 理解惠斯通电桥的基本原理，掌握用直流单臂电桥测量电阻的方法。
2. 了解用双臂电桥测量低值电阻的原理，初步掌握直流双臂电桥的使用方法。
3. 理解直流非平衡电桥的工作原理和线性条件，掌握用非平衡电桥测量其他物理量的基本方法。

## 实验原理

电阻是常用的电学元器件，常用电桥进行阻值的精密测量。电桥最初由英国发明家克里斯蒂（Samuel Hunter Christie）于 1833 年发明。惠斯通（Charles Wheatstone）于 1843 年设计了现在被称为惠斯通电桥的电路，并首次用它测量电阻。1860 年西门子（E. W. von Siemens）使用水银柱作为电阻标准并用惠斯通电桥测量，使该电桥流行起来。1861 年汤姆孙（开尔文勋爵）设计了可测量极小电阻的电桥，称为开尔文电桥（Kelvin Bridge）。

电桥主要分为两类：

- **单臂电桥**（即惠斯通电桥）：一般用于精确测量中值电阻（\(1\sim 10^{5}\,\Omega\)）。
- **双臂电桥**（即开尔文电桥）：主要适用于测量低电阻（\(10^{-6}\sim 10\,\Omega\)）。

### 1. 单臂电桥的线路原理

单臂电桥的基本线路如图 7-1 所示，由四个电阻 \(R_1\)、\(R_2\)、\(R_x\)、\(R_s\) 和检流计等构成。接入检流计（平衡指示）的对角线称为“桥”，四个电阻称为“桥臂”。

!!! note “仪器/实物图：单臂电桥基本线路”
    页2, 图7-1 单臂电桥基本线路

一般情况下，桥路上检流计中有电流通过，指针偏转。若适当调节某一电阻值（例如改变 \(R_s\)），可使 C、D 两点电位相等，此时流过检流计 G 的电流 \(I_g=0\)，称为**电桥平衡**。则有：

\[
U_{AC}=U_{AD} \tag{7-1}
\]

\[
U_{CB}=U_{DB} \tag{7-2}
\]

\[
I_1=I_x,\quad I_2=I_s \tag{7-3}
\]

由欧姆定律，平衡时：

\[
I_1 R_1 = I_2 R_2 \tag{7-4}
\]

\[
I_x R_x = I_s R_s \tag{7-5}
\]

由以上两式可得：

\[
\boxed{\,R_x=\dfrac{R_1}{R_2}\,R_s\,} \tag{7-6}
\]

式 (7-6) 即为电桥的平衡条件。若 \(R_1\)、\(R_2\) 已知，即可由式 (7-6) 求出 \(R_x\)。通常取 \(R_1\)、\(R_2\) 为标准电阻，称为**比率臂**，将 \(R_1/R_2\) 称为**桥臂比**；\(R_s\) 为可调电阻，称为**比较臂**。改变 \(R_s\) 使电桥达到平衡（检流计 G 中无电流），便可测出被测电阻 \(R_x\) 之值。

### 2. 用交换法减小和消除系统误差

用单臂电桥测量 \(R_x\) 的误差，除其他因素外，还与标准电阻 \(R_1\)、\(R_2\) 的误差有关。可采用**交换法**消除这一系统误差：

1. 先连接好电桥电路，调节 \(R_s\) 使 G 中无电流，由式 (7-6) 求出 \(R_x\)；
2. 将 \(R_x\) 与 \(R_s\) 交换位置，再调节 \(R_s\) 使 G 中无电流，记下此时的 \(R_s'\)，可得：

\[
R_x=\dfrac{R_2}{R_1}\,R_s' \tag{7-7}
\]

式 (7-6) 和式 (7-7) 相乘得：

\[
\boxed{\,R_x=\sqrt{R_s\cdot R_s'}\,} \tag{7-8}
\]

这样就消除了由 \(R_1\)、\(R_2\) 本身的误差引入的对 \(R_x\) 的测量误差。此时 \(R_x\) 的测量误差只与电阻箱 \(R_s\) 的仪器误差有关，而 \(R_s\) 可选用高精度的标准电阻箱，系统误差即可减小。

> **重点**：交换法是消除比率臂误差的经典方法，测量结果 \(R_x=\sqrt{R_s R_s'}\) 只依赖比较臂两次读数，与 \(R_1\)、\(R_2\) 的具体阻值无关。

### 3. 电桥的灵敏度

式 (7-6) 是在电桥平衡条件下推导出来的，而电桥是否平衡实际上是根据检流计是否有偏转来判断的。检流计的灵敏度总是有限的，实验中所用检流计指针偏转一格所对应的电流大约为 \(10^{-6}\,\text{A}\)。当通过它的电流比 \(10^{-7}\,\text{A}\) 还要小时，指针偏转小于 0.1 格，就很难觉察出来。

假设电桥在 \(R_1/R_2=1\) 时调到平衡，则有 \(R_x=R_s\)。这时若把 \(R_s\) 改变 \(\Delta R_s\)，电桥失去平衡，检流计中有电流 \(I_g\) 流过。但如果 \(I_g\) 小到使检流计觉察不出来，还会认为电桥还是平衡的，因而得出 \(R_x=R_s+\Delta R_s\)。这样就会因为检流计的反应不够灵敏而带来测量误差 \(\Delta R_x=\Delta R_s\)。为表示此误差对测量结果影响的严重程度，引入**电桥灵敏度**的概念，定义为：

\[
\boxed{\,S=\dfrac{\Delta n}{\Delta R_s/R_s}\,} \tag{7-9}
\]

式中 \(\Delta R_s\) 是在电桥平衡后 \(R_s\) 的微小改变量（实际上改变任一臂所得出的电桥灵敏度是一样的）；\(\Delta n\) 是由于电桥偏离平衡而引起的检流计偏转格数。\(S\) 越大说明电桥越灵敏，带来的误差也就越小。

举例来说，检流计指针有 1/5 格的偏转时即可以觉察出来，如果 \(S=100\) 格，则只要 \(R_s\) 改变 0.2% 就可以觉察到了，在这种情况下由于电桥灵敏度的限制所带来的误差肯定小于 0.2%。

\(S\) 的定义式可变换为：

\[
S=S_i\cdot S_l \tag{7-10}
\]

式中 \(S_i\) 为检流计的电流灵敏度，\(S_l\) 为电桥线路的灵敏度。即电桥的灵敏度不仅与检流计的灵敏度有关，而且还与线路参数（\(R_1\)、\(R_2\)、\(R_s\) 等）的取值有关。

> **易错**：电桥灵敏度 \(S\) 是有量纲的（单位“格”），它由检流计灵敏度和线路灵敏度共同决定，并非只取决于检流计。改变桥臂比和桥臂电阻取值都会影响 \(S\)。

一般在用电桥测电阻时，应保证较高的电桥灵敏度。在检流计、电源一定的情况下，桥臂比及桥臂电阻的取值都会影响电桥的灵敏度。同时，要合理确定桥臂比 \(R_1/R_2\) 之值，使测量结果的有效数字位数足够多（一般应比由误差决定的位数多一位）。但在测量时，还应保证在改变 \(R_s\) 的最小可调挡（\(\Delta R_s\)）两次，或改变量为仪器误差 \(\Delta I\) 时，应能觉察出检流计指针的偏转（不小于 0.2 格），否则位数再多也不是实际的。

### 4. 接触电阻和导线电阻的影响

不同阻值的电阻在精确测量时会遇到不同的特殊问题。用单臂电桥测中值电阻时，能较好地克服电表准确度及电表内阻给线路带来的系统误差，但测量低电阻时无法解决导线电阻和接点处的接触电阻（统称“附加电阻”）对测量的影响。

附加电阻一般在 \(10^{-2}\sim 10^{-5}\,\Omega\) 数量级。如果待测电阻是 \(0.01\,\Omega\)，而附加电阻为 \(0.001\,\Omega\) 时，其影响可达 10%。如果所测电阻在 \(0.001\,\Omega\) 以下，就无法再用单臂电桥来测此低电阻，必须对线路加以改进。双臂电桥就是对单臂电桥加以改进而成的，它适用于 \(10^{-6}\sim 10\,\Omega\) 电阻的测量。

### 5. 用四端式电阻来减小附加电阻的影响

图 7-2 为研究附加电阻对低电阻测量影响的双臂电桥原理图。

!!! note “仪器/实物图：双臂电桥原理图（附加电阻影响）”
    页3, 图7-2 双臂电桥原理图

图中 \(r_1\)、\(r_2\)、\(r_3\) 表示与接点 a 相连的三条支路的附加电阻，\(r_4\)、\(r_5\)、\(r_6\) 表示与接点 b 相连的三条支路的附加电阻。其中 \(r_1\)、\(r_6\) 与电流测量回路或供电回路串联，因为它们很小，所以它们的串入对电路状态不会产生太大影响；\(r_3\)、\(r_4\) 与电压测量回路串联，与 \(R_x\) 并联，它们的接入相当于加大了电压表内阻，对测量是有益的；\(r_2\) 和 \(r_5\) 与 \(R_x\) 串联，是对 \(R_x\) 测量直接有影响的部分。因此，为了减少附加电阻对 \(R_x\) 测量的影响，就应尽量减小 \(r_2\) 和 \(r_5\)，而相对地加大 \(r_3\) 和 \(r_4\)。

图 7-3 为四端式电阻等效原理图。如把 \(R_x\) 的接头改成“四端”，在接线时将与电流回路相连的接点 A、B（称为“电流接头”）及与电压回路相连的接点 A'、B'（称为“电压接头”）分开，把电压接头放在里边，并用两较大的纯铜接线柱（为了减小接触电阻）作为电流接头，用两较小的接线柱作为电压接头，在电压接头与电流接头之间用粗的纯银线或纯铜片连接以减小接线电阻。那么 \(r_2\) 和 \(r_5\) 就可能减得很小了，此时两电压接头间的电阻就能较准确地表示为 \(R_x\) 了。

!!! note “仪器/实物图：四端式电阻等效原理图”
    页4, 图7-3 四端式电阻等效原理图

> **重点**：凡是要求有比较准确阻值的低电阻，一定是“四端”式接线的电阻，其阻值就是两电压接头 A'、B' 之间的电阻。电流接头在外侧（用粗铜柱），电压接头在内侧（用小接线柱），中间用粗导线/铜片连接。

### 6. 双臂电桥工作原理

双臂电桥的线路如图 7-4 所示，等效电路如图 7-5 所示。它有两大特点：

!!! note “仪器/实物图：双臂电桥线路图”
    页4, 图7-4 双臂电桥线路图

!!! note “仪器/实物图：双臂电桥等效电路”
    页4, 图7-5 双臂电桥等效电路

1. **待测电阻 \(R_x\) 和比较臂电阻 \(R_s\) 都采用四端接法接入电路。** 三根电流端引线的附加电阻分别为 \(r\)、\(r_1\)、\(r_6\)。其中 \(r\) 包括导线电阻、接点接触电阻以及两电压接头间电阻的总和；\(r_1\) 和 \(r_6\) 也是类似情况。另外，四根电压端引线的附加电阻分别为 \(r_2\)、\(r_3\)、\(r_4\)、\(r_5\)，它们都包含导线电阻和接触电阻。

2. **在电路中增加了 \(R_3\) 和 \(R_4\) 两个电阻，即多了一组桥臂。** 由于有两组桥臂，故称为双臂电桥。

双臂电桥可减小附加电阻对测量低电阻的影响，原因有三：

- 一是 \(R_x\) 和 \(R_s\) 均采用了四端接法，巧妙地避免了接线电阻和导线电阻对测量电阻的影响（这里并不是说它们被消除了，而是被引到其他支路上去了，在其他支路上它们往往可以被忽略不计）；
- 二是桥臂电阻分别比相应的附加电阻大得多，附加电阻也可忽略不计；
- 三是 \(R_x\) 和 \(R_s\) 采用足够粗的导线连接，使得附加电阻（又称跨线电阻）\(r\) 很小，又由于四个桥臂电阻 \(R_1\)、\(R_2\)、\(R_3\)、\(R_4\) 比附加电阻大得多，于是当双臂电桥平衡时，桥臂电流 \(i_1\) 和 \(i_2\) 必然比流过 \(R_x\) 和 \(R_s\) 的电流 \(I\) 小得多。这样，附加电阻 \(r_2\)、\(r_3\)、\(r_4\)、\(r_5\) 上的电压降与四个桥臂电阻以及 \(R_x\)、\(R_s\) 上的电压降相比小得多，因而可忽略不计。

适当调节电阻 \(R_1\)、\(R_2\)、\(R_3\)、\(R_4\) 使检流计 G 没有电流通过，即电桥达到平衡。此时流过 \(R_1\) 和 \(R_2\)、\(R_3\) 和 \(R_4\)、以及 \(R_x\) 和 \(R_s\) 的电流分别相等，设分别为 \(i_1\)、\(i_2\)、\(I\)。当双臂电桥平衡时，两个电压接头（检流计两端）电位相等，下述关系式成立：

\[
\begin{cases}
(R_1+r_2)\,i_1 = R_x\,I + (R_3+r_3)\,i_2 \\
(R_2+r_5)\,i_1 = R_s\,I + (R_4+r_4)\,i_2 \\
r\,(I-i_1-i_2) = (R_3+R_4+r)\,i_2
\end{cases}
\tag{7-11}
\]

> **注**：式 (7-11) 第三式中 \(r\) 为跨线电阻，\((I-i_1-i_2)\) 为跨线上的电流。

为了使附加电阻 \(r_2\)、\(r_3\)、\(r_4\)、\(r_5\) 的影响可以忽略不计，在双臂电桥电路设计中要求桥臂电阻足够大，即 \(R_1\gg r_2\)、\(R_2\gg r_5\)、\(R_3\gg r_3\)、\(R_4\gg r_4\)。同时 \(R_x\) 和 \(R_s\) 的连接采用一根粗导线，使得附加电阻 \(r\) 很小，以满足 \(I\gg i_1\) 和 \(I\gg i_2\) 的条件。于是式 (7-11) 可简化为：

\[
\begin{cases}
R_1\,i_1 = R_x\,I + R_3\,i_2 \\
R_2\,i_1 = R_s\,I + R_4\,i_2 \\
r\,I = (R_3+R_4+r)\,i_2
\end{cases}
\tag{7-12}
\]

解此方程组可得：

\[
\boxed{\,R_x=\dfrac{R_1}{R_2}\,R_s+\dfrac{r\,R_4}{R_3+R_4+r}\left(\dfrac{R_1}{R_2}-\dfrac{R_3}{R_4}\right)\,} \tag{7-13}
\]

在实验测量过程中，若始终保持 \(R_1/R_2=R_3/R_4\)，则式 (7-13) 中的第二项会始终保持为零，这样双臂电桥测低电阻的结果表达式与单臂电桥完全一样，即：

\[
\boxed{\,R_x=\dfrac{R_1}{R_2}\,R_s\,} \tag{7-14}
\]

> **重点**：双臂电桥消除附加电阻影响的关键在于：① 四端接法把附加电阻引到无关支路；② 桥臂电阻远大于附加电阻；③ 保持 \(R_1/R_2=R_3/R_4\) 使式 (7-13) 修正项为零。实际仪器中 \(R_1\) 与 \(R_3\)、\(R_2\) 与 \(R_4\) 常采用同步调节的联动旋钮，以保证比值始终相等。

### （1）自组直流单臂电桥

按图 7-6 自组直流单臂电桥电路。选择合适的倍率测量未知电阻阻值，测出电桥的灵敏度，分析直流单臂电桥比率臂阻值、检流计灵敏度对电桥灵敏度及电阻测量精度的影响。

!!! note “仪器/实物图：自组直流单臂电桥电路”
    页5, 图7-6 自组直流单臂电桥电路

> **易错**：连线操作时应遵从电学实验操作规程，连线按回路依次连接，并使电路布局合理，学会排查和解决直流单臂电桥电路故障。

图中“桥”路开关 \(S_c\) 上并联了一个高电阻 \(R_k\)，其作用是保护检流计，方便平衡状态的调节。测量时先打开 \(S_c\)，由于 \(R_k\) 较大，所以流经检流计的电流不会很大；调节 \(R_s\) 使电桥接近于平衡状态时，再合上 \(S_c\) 使 \(R_k\) 短路，此时桥路的灵敏度增高，再仔细调节 \(R_s\)，使电桥平衡（检流计 G 的指针指零）。

滑线变阻器 \(R_n\) 的作用与 \(R_k\) 类似：测量时先将滑动头置于最左端，由于 \(R_n\) 较大，干路中电流不会太大，流经检流计的电流也受到一定限制；调节 \(R_s\) 使电桥接近平衡时，再将滑动头移向右端，增大干路电流以提高桥路的灵敏度，然后再仔细调节 \(R_s\) 使电桥平衡。

- 用**交换法**进行系统误差研究：将 \(R_x\) 与 \(R_s\) 交换位置重测，用式 (7-8) 计算 \(R_x\)，求出不确定度，并与上面的测量结果做分析比较。
- **测定电桥灵敏度**，并计算或测定由电桥灵敏度引入的误差（取人眼能够判断偏转的极限为 0.2 格）。

### （2）用双臂电桥测金属电阻率 \(\rho\)

按图 7-4 正确连线，注意 \(R_x\)、\(R_s\) 均采用四端接法。实验连线时，要考虑哪些部分要用短而粗的导线，哪些部分可以不做要求。选取合适的 \(R_1/R_2=R_3/R_4\) 比值及各电阻的阻值，测定待测低电阻 \(R_x\) 之值，计算不确定度并正确表达测量结果。用不同的桥臂比测量 \(R_x\)，并分析结果的有效数字与桥臂比选取的关系。

改装直流单臂电桥为非平衡电桥，研究非平衡电桥的线性条件。

### （4）测量热敏电阻的温度特性

用电桥测量热敏电阻在不同温度下的电阻值，用作图法（曲线改直）及回归法确定电阻值随温度变化的经验公式。由物理理论已知此经验公式的函数形式为：

\[
R_T=A\,e^{B/T}
\]

其中 \(T\) 为热力学温度。

---

## 数据处理

### 1. 单臂电桥测中值电阻

**直接测量法**：由式 (7-6)

\[
R_x=\dfrac{R_1}{R_2}\,R_s
\]

**交换法**：由式 (7-8)

\[
R_x=\sqrt{R_s\cdot R_s'}
\]

其中 \(R_s\)、\(R_s'\) 为交换前后比较臂的两次读数。交换法消除了比率臂 \(R_1\)、\(R_2\) 误差的影响，测量结果只与 \(R_s\) 的仪器误差有关。

**不确定度分析**：

- 交换法结果 \(R_x=\sqrt{R_s R_s'}\) 的相对不确定度：

\[
\dfrac{u(R_x)}{R_x}=\dfrac{1}{2}\sqrt{\left(\dfrac{u(R_s)}{R_s}\right)^2+\left(\dfrac{u(R_s')}{R_s'}\right)^2}
\]

- 电阻箱 \(R_s\) 的仪器误差由其准确度等级 \(a\) 决定：

\[
\Delta_{\text{仪}}(R_s)=R_s\cdot a\%+\Delta R_0
\]

其中 \(\Delta R_0\) 为零电阻误差（通常取最小步进值）。仪器误差限可作为 B 类不确定度 \(u_B=\Delta_{\text{仪}}/\sqrt{3}\)。

### 2. 电桥灵敏度及其引入的误差

由式 (7-9)，电桥灵敏度：

\[
S=\dfrac{\Delta n}{\Delta R_s/R_s}
\]

由灵敏度限制引入的测量误差（取人眼判断极限 \(\Delta n_0=0.2\) 格）：

\[
\Delta R_x^{(S)}=\dfrac{\Delta n_0}{S}\,R_x=\dfrac{0.2}{S}\,R_x
\]

对应的相对误差为 \(\Delta n_0/S=0.2/S\)。例如 \(S=100\) 格时，该项相对误差为 0.2%。

> **易错**：灵敏度引入的误差 \(\Delta R_x^{(S)}\) 与电阻箱仪器误差 \(\Delta_{\text{仪}}\) 是两类独立误差，合成总不确定度时应取方和根：\(u(R_x)=\sqrt{u_B^2+u_S^2}\)，其中 \(u_S=\Delta R_x^{(S)}/\sqrt{3}\)（按均匀分布）或直接取 \(\Delta R_x^{(S)}\) 作为限值。

### 3. 双臂电桥测低电阻

由式 (7-14)，在保持 \(R_1/R_2=R_3/R_4\) 条件下：

\[
R_x=\dfrac{R_1}{R_2}\,R_s
\]

不确定度主要来源于比较臂 \(R_s\) 的仪器误差及灵敏度限制，分析方法与单臂电桥类似。

**金属电阻率**：测得低电阻 \(R_x\) 后，由电阻定律计算电阻率：

\[
\rho=\dfrac{R_x\,S}{L}=\dfrac{R_x\,\pi d^2}{4L}
\]

其中 \(L\) 为两电压接头之间的长度，\(d\) 为金属棒直径（用螺旋测微器多次测量取平均），\(S\) 为横截面积。相对不确定度：

\[
\dfrac{u(\rho)}{\rho}=\sqrt{\left(\dfrac{u(R_x)}{R_x}\right)^2+\left(\dfrac{2u(d)}{d}\right)^2+\left(\dfrac{u(L)}{L}\right)^2}
\]

### 4. 热敏电阻温度特性

已知 \(R_T=A\,e^{B/T}\)，两边取对数：

\[
\ln R_T=\ln A+\dfrac{B}{T}
\]

即 \(\ln R_T\) 与 \(1/T\) 成线性关系。

- **作图法（曲线改直）**：以 \(1/T\) 为横坐标、\(\ln R_T\) 为纵坐标作图，直线的斜率为 \(B\)，截距为 \(\ln A\)，从而定出 \(A\)、\(B\)。
- **回归法**：对 \((1/T,\,\ln R_T)\) 数据作最小二乘线性回归，由斜率和截距确定 \(B\) 和 \(A\)，并给出相关系数以检验线性程度。

### 5. 有效数字与桥臂比的选取

测量结果的有效数字位数由电桥灵敏度决定。改变 \(R_s\) 最小步进挡两次（或改变量为仪器误差）时，检流计偏转应不小于 0.2 格，否则有效数字位数再多也不实际。桥臂比 \(R_1/R_2\) 的选取应使比较臂 \(R_s\) 的各挡均能投入使用，以获得足够多的有效数字位数（一般比误差决定的位数多一位）。

## 注意事项

1. **保护检流计**：实验时应注意先合电源开关 \(S\) 后合桥路开关 \(S_c\)，断开时先断 \(S_c\) 后断 \(S\)。
2. **保护电阻调节**：开始通电时，保护电阻 \(R_k\)（及滑线变阻器 \(R_n\)）应放在最大位置，待基本调平后，再减小 \(R_k\)（移动滑线变阻器滑动头）重新调平衡，以提高灵敏度。
3. **间歇通电**：电源开关及检流计按钮应间歇使用，不能长时间接通，以免元件发热漂移或损坏。

> **易错**：通电顺序“先电源后桥路、断电先桥路后电源”必须严格遵守，否则检流计可能因不平衡的大电流冲击而损坏。

## 思考题

### 预习思考题

1. 电桥由哪几部分组成？电桥平衡的条件是什么？具体操作时如何实现电桥平衡？

    ??? note “参考答案”
        电桥由四个桥臂电阻、检流计（桥）、电源组成。平衡条件为对臂电阻乘积相等 \(R_1 R_s = R_2 R_x\)（即检流计电流 \(I_g = 0\)）。操作时先粗调（保护电阻 \(R_k\) 置大，限制电流），调节 \(R_s\) 使检流计接近指零；再细调（合上 \(S_c\) 短路保护电阻，提高灵敏度），仔细调 \(R_s\) 至检流计准确指零。

2. 电阻箱的误差如何决定？

    ??? note “参考答案”
        电阻箱的仪器误差由其准确度等级 \(a\) 决定，公式为 \(\Delta_{\text{仪}} = R \cdot a\% + \Delta R_0\)，其中 \(R\) 为读数，\(\Delta R_0\) 为零电阻误差（通常取最小步进值）。仪器误差限可作为 B 类不确定度 \(u_B = \Delta_{\text{仪}}/\sqrt{3}\)。

3. 双臂电桥与单臂电桥有哪些异同？

    ??? note “参考答案”
        相同点：都是利用电桥平衡条件（对臂电阻乘积相等）测量电阻，平衡时检流计指零。不同点：（1）单臂电桥有四个桥臂，双臂电桥有六个桥臂（多一组 \(R_3\)、\(R_4\)）；（2）单臂电桥适用于中值电阻（\(1\sim10^5\,\Omega\)），双臂电桥适用于低电阻（\(10^{-6}\sim10\,\Omega\)）；（3）单臂电桥无法消除接触电阻和导线电阻的影响，双臂电桥通过四端接法和同步调节 \(R_1/R_2 = R_3/R_4\) 消除附加电阻影响；（4）双臂电桥的待测电阻和比较臂电阻均采用四端接法。

4. 在双臂电桥线路中是怎样消除导线电阻和接触电阻影响的？试简要说明之。

    ??? note “参考答案”
        通过三方面消除：（1）四端接法将附加电阻引入电流回路或高阻桥臂支路，使其可忽略不计；（2）桥臂电阻远大于附加电阻，附加电阻的影响相对极小；（3）保持 \(R_1/R_2 = R_3/R_4\) 使式 (7-13) 修正项为零，同时用粗短导线减小跨线电阻 \(r\)。

### 思考题

1. 下列因素是否会增大电桥测量误差？
   1. 电源电压不太稳定。
   2. 忽略导线电阻。
   3. 检流计没有校好零点。
   4. 检流计灵敏度不高。
   5. 桥臂电阻取值过大。

    ??? note “参考答案”
        1. 电源电压不稳定会增大误差。电压波动导致电桥工作点漂移，平衡判断不准确。
        2. 忽略导线电阻会增大误差。对于低电阻测量，导线电阻与被测电阻可比拟，不可忽略（单臂电桥）；双臂电桥可消除此影响。
        3. 检流计未校好零点会增大误差。零点偏移导致平衡判断错误，引入系统误差。
        4. 检流计灵敏度不高会增大误差。灵敏度不足时，微小不平衡无法察觉，导致测量精度下降。
        5. 桥臂电阻取值过大会增大误差。桥臂电阻过大时桥路电流减小，检流计偏转变小，电桥灵敏度降低。

2. 当电桥达到平衡后，互换电源与检流计的位置，电桥是否仍保持平衡？为什么？

    ??? note “参考答案”
        电桥仍保持平衡。因为电桥平衡条件为对臂电阻乘积相等（\(R_1 R_s = R_2 R_x\)），与电源、检流计所在对角线的位置无关。平衡仅由四个桥臂的阻值关系决定，互换电源与检流计只是改变了两个对角线的角色，不影响四臂之间的比例关系。

3. 若同学用图 7-6 所示电桥测电阻 \(R_x\)，出现下列现象，试解释之。
   1. 无论怎样调节 \(R_s\)，检流计的指针始终偏在零点的一侧。
   2. 调节 \(R_s\) 的最小挡，检流计的指针不是偏左就是偏右，始终不能准确指零。

    ??? note “参考答案”
        1. 指针始终偏一侧，说明某桥臂断路或短路、接线错误、或电源极性使某支路始终不平衡。应检查各臂是否连通、阻值是否正常、接线是否正确。
        2. 调最小挡指针左右摆不能指零，说明电桥灵敏度不足或最小步进太大。应提高电源电压或检流计灵敏度，或选择更合适的桥臂比使 \(R_s\) 的有效数字位数匹配。

4. 能否用电桥测定电流表的内阻？如能，试画出电路图。

    ??? note “参考答案”
        可以。将电流表作为待测电阻 \(R_x\) 接入电桥的一个桥臂。由于电流表本身可指示电流，可将检流计位置用电流表替代，或仍用检流计。电桥平衡时，电流表读数为零（或最小），由 \(R_x = (R_1/R_2) R_s\) 求出内阻。注意：通过电流表的电流不能超过其量程，应串联保护电阻并降低电源电压。

5. 用双臂电桥测低电阻时，实验中应保证满足哪些测量条件？为什么？

    ??? note “参考答案”
        应保证：（1）\(R_1/R_2 = R_3/R_4\)（同步调节），使式 (7-13) 修正项为零；（2）桥臂电阻远大于附加电阻，使附加电阻可忽略；（3）\(R_x\) 和 \(R_s\) 间的跨线电阻 \(r\) 足够小（用粗短导线连接），满足 \(I \gg i_1\) 和 \(I \gg i_2\)；（4）采用四端接法，电流接头在外侧、电压接头在内侧。这些条件确保附加电阻对测量的影响可忽略，使双臂电桥公式简化为 \(R_x = (R_1/R_2) R_s\)。

6. 如果发现双臂电桥灵敏度不足，原则上可采取哪些措施？这些措施又受到什么限制？

    ??? note “参考答案”
        可采取的措施及限制：（1）提高电源电压——受桥臂电阻额定功率和待测电阻允许电流限制；（2）提高检流计灵敏度——受检流计内阻与桥路匹配及抗干扰能力限制；（3）减小桥臂电阻——受桥臂电阻须远大于附加电阻的条件限制；（4）减小跨线电阻——受导线材料和截面积限制。各措施需综合考虑，不能无限制提高。

7. 如何正确选择 \(R_1/R_2=R_3/R_4\) 的比值，以保证足够的有效数字？

    ??? note “参考答案”
        桥臂比 \(R_1/R_2\) 的选取应使比较臂 \(R_s\) 的各挡均能投入使用，以获得足够多的有效数字位数（一般比误差决定的位数多一位）。具体原则：根据 \(R_x\) 的估计值，选择桥臂比使 \(R_s = (R_2/R_1) R_x\) 落在电阻箱可读范围的中段（使最高位和最低位挡均有读数）。同时应保证改变 \(R_s\) 最小步进挡两次时，检流计偏转不小于 0.2 格，否则有效数字位数再多也不实际。
