## 实验目的

1.  理解牛顿环和劈尖干涉等厚干涉原理。
2.  掌握读数显微镜的测量方法。
3.  应用牛顿环的干涉计量方法测量透镜的曲面参量（平凸透镜的曲率半径 \(R\)）。
4.  利用劈尖干涉测量金属细丝的直径。

---

## 实验原理

光的干涉是最重要的光学现象之一。早在1675年，牛顿在制作天文望远镜时，偶然将一个望远镜的物镜放置在一块平板玻璃上，发现了牛顿环这一干涉现象。将两个倾斜的透明平板叠放在一起时，光通过两个平板的间隙也会产生干涉图样，这种现象称为劈尖干涉。日常生活中也能见到诸如肥皂泡呈现的五颜六色、雨后路面上油膜的多彩图样等光的等厚干涉现象。

### 1. 牛顿环

当一个曲率半径较大的平凸透镜置于一块光学平面玻璃上时，如图18-1所示，在透镜和平板玻璃之间形成一层空气薄膜，其厚度自中心向边缘逐渐增加，且同一圆周上空气层的厚度相同。当用平行单色光自透镜上方垂直照射时，入射光在空气层的上下表面反射后相遇。由于其光程差恒定，因此产生干涉现象。显然，随着厚度的不同，它们的干涉图样将以接触点为中心，呈现一系列明暗相间的同心圆环——牛顿环，如图18-2所示。

!!! note "原理示意图：牛顿环构造及光路图"
    页2，牛顿环构造及光路图：平凸透镜置于平板玻璃上，单色光垂直入射，空气层上下表面反射光干涉

由光路图18-1可知，与第 \(k\) 级干涉环对应的两束光的光程差为

\[
\delta = 2 n_0 d + \frac{\lambda}{2} \tag{18-1}
\]

式中，\(n_0\) 为薄膜的折射率，该实验中薄膜为空气层，取 \(n_0 \approx 1\)；\(\lambda/2\) 是因光线由光疏介质进入光密介质在反射时产生半波损失而附加的光程差。

由几何关系得

\[
r^2 = R^2 - (R - d)^2 = 2Rd - d^2 \tag{18-2}
\]

忽略二阶小量，有

\[
d = \frac{r^2}{2R} \tag{18-3}
\]

将式 (18-3) 代入式 (18-1) 得到

\[
\delta = \frac{r^2}{R} + \frac{\lambda}{2} \tag{18-4}
\]

由干涉条件可知，当

\[
\delta = \frac{r^2}{R} + \frac{\lambda}{2} = (2k+1)\frac{\lambda}{2} \tag{18-5}
\]

时，干涉条纹为暗纹，所以第 \(k\) 级暗环的半径为

\[
r_k = \sqrt{k \lambda R} \quad (k = 0, 1, 2, \ldots) \tag{18-6}
\]

由式 (18-6) 可知，如果单色光源的波长 \(\lambda\) 已知，测得第 \(k\) 级暗环的半径 \(r_k\)，即可算出曲率半径 \(R\)；反之，若 \(R\) 已知，测得 \(r_k\) 后，即可计算出入射单色光源的波长 \(\lambda\)。

!!! note "仪器/实物图：牛顿环干涉条纹"
    页2，牛顿环干涉条纹照片：明暗相间的同心圆环

> **重点**：由于平凸透镜的凸面和平板玻璃的平面不可能实现理想的点接触，因此直接使用测量半径 \(r_k\) 的公式时往往误差较大。接触压力会引起局部形变，使接触处成为一个圆面，造成牛顿环中心为一暗斑；或者由于凸面和平面的接触处存在灰尘，附加了光程差，使得牛顿环中心并非一个几何暗点，而是一个不清晰的亮（暗）圆斑。此时 \(r_k\) 不易测准。为克服这一缺点，采用测量干涉圆环直径的方法（实际上测得的是干涉环的弦长）。

### 2. 最小二乘法数据处理

测出一系列对应各级（\(k\) 为变量）暗环的直径 \(D_k\)（实验中用弦长代替直径），根据实验原理，相关物理量满足关系：

\[ D_{k}^2 = 4 k \lambda R, \quad D_{k+m}^2 = 4(k+m) \lambda R \]

因而

\[
D_{k+m}^2 - D_k^2 = 4 m R \lambda \tag{18-7}
\]

如果实验测量第20环到第30环暗环的弦长，可记 \(k_0 = 20\)，\(y_m = (D_{k_0+m}^2 - D_{k_0}^2) / (4\lambda)\)，\(x = m\)（\(m = 1, 2, 3, \ldots\)），则实验数据应满足线性关系：

\[ y = R x + a_0 \]

利用最小二乘法，平凸透镜曲率半径 \(R\) 为上述直线方程的斜率值，即

\[
R = \frac{n \sum x_i y_i - \sum x_i \sum y_i}{n \sum x_i^2 - \left(\sum x_i\right)^2} \tag{18-8}
\]

进一步地，可根据最小二乘法计算斜率的不确定度，进而得到平凸透镜实验测量值的不确定度，并可分析误差来源。

### 3. 劈尖干涉

如图18-3所示，在两片相互叠合的平面玻璃板之间，靠近一端处垫入一金属细丝（或薄纸片），则两玻璃板间的空气层形成一个空气劈尖。两玻璃片的交线称为棱边，在平行于棱边的直线上，劈尖空气膜的厚度是相等的。当平行单色光垂直入射时，空气劈尖上、下表面反射的两束光线将形成相干光，它们在劈尖的上表面相遇而产生干涉，呈现出一组与棱边平行的明暗相间的直线条纹，如图18-4所示。

!!! note "原理示意图：劈尖构造图"
    页3，劈尖构造图：两片玻璃板间垫入金属丝形成空气劈尖，单色光垂直入射

!!! note "仪器/实物图：劈尖干涉条纹"
    页3，劈尖干涉条纹：与棱边平行的明暗相间直条纹

由光路可知，空气劈尖下、上表面反射的两束光的光程差为

\[
\delta = 2d + \frac{\lambda}{2} \tag{18-9}
\]

式中，\(d\) 为干涉处空气劈尖膜的厚度；\(\lambda/2\) 是因光线由光疏介质进入光密介质在反射时产生半波损失而附加的光程差。

由干涉条件可知：
- 当 \(\delta = k\lambda\)（\(k = 1, 2, \ldots\)）时，为亮条纹；
- 当 \(\delta = (2k+1)\lambda/2\)（\(k = 0, 1, 2, \ldots\)）时，为暗条纹。

两相邻暗纹（或亮纹）对应的劈尖厚度之差为

\[
\Delta d = d_{k+1} - d_k = \frac{\lambda}{2} \tag{18-10}
\]

若两暗纹之间的距离为 \(l\)，则劈尖的夹角 \(\theta\) 可利用下式求得：

\[
\tan \theta = \frac{\lambda}{2l} \tag{18-11}
\]

设金属细丝（或薄纸片）至棱边的距离为 \(L\)，则金属细丝的直径（或薄纸片的厚度）为

\[
D = \frac{L \lambda}{2l} \tag{18-12}
\]

这就是本实验利用劈尖干涉测量金属细丝直径（或薄纸片厚度）的公式。

> **重点**：实验上往往不是测量两条相邻条纹的间距，而是测量相差 \(N\) 级（例如50级或100级）的两条暗纹的间距，从而使测量结果更为准确。此时式 (18-12) 中 \(l\) 应替换为 \(N\) 条条纹总间距除以 \(N\)，即 \(l = s / N\)，则 \(D = L\lambda N / (2s)\)。

**读数显微镜简介**：

一般的显微镜仅具有放大物体的作用，不能测量物体的线度。如果在显微镜的目镜中装上十字叉丝，并将镜筒固定在一个可以左右或上下移动的托板上，而托板移动的距离则由千分尺读出。这样改装的显微镜称为读数显微镜，它是用于精确测量直线长度的光学仪器。

读数显微镜的主要部分为放大待测物体用的显微镜和读数用的主尺及副尺。显微镜由目镜、物镜和十字叉丝组成，使用时镜筒可以处于垂直于水平位置，也可以将基座旋转90°，使显微镜镜筒呈水平状态。旋转测微鼓轮，可使显微镜镜筒沿导轨横向移动。

> **重点**：测微鼓轮每旋转一周，显微镜镜筒移动1 mm，镜筒的移动量从附在导轨上的50 mm直尺上读出。测微鼓轮圆周均分100个刻度，因此测微鼓轮每旋转一格，显微镜镜筒移动0.01 mm。

## 实验步骤

### 一、用牛顿环测量平凸透镜的曲率半径

**1. 实验装置的调整**

以钠光灯为光源，其发出的光经平板玻璃片反射后垂直射入牛顿环上，形成的干涉环可通过读数显微镜进行观察和测量。观察并记录牛顿环等厚干涉现象，并从理论上给出解释。

**2. 测量平凸透镜的曲率半径**

1.  转动读数显微镜读数鼓轮，同时在目镜中观察，使十字叉丝由牛顿环中央缓慢向一侧移动至第35环。
2.  从第35环起**单方向**移动十字叉丝，测出第30环到第20环的直径的左右两边读数，求得第20～30环的直径（弦长）。
3.  将所测得的数据用回归法（最小二乘法）或图解法进行处理，求出平凸透镜的曲率半径 \(R\)。

> **易错**：测量过程中若读数鼓轮反方向移动则存在空程差，鼓轮只能单方向移动。

**3. 用白光观察**

用白炽灯代替钠光灯，观察白光照明下的牛顿环干涉图样，与钠光下观察的图样进行比较，并从理论上给出解释。

### 二、用劈尖干涉测量金属细丝的直径

1.  安装好劈尖装置后将其放置在读数显微镜的载物台上，点亮钠光灯，调节反光玻璃板和显微镜，直至观察到清晰的干涉条纹和十字叉丝。
2.  测量50或100条暗纹之间的距离 \(s\) 和金属丝到棱边的距离 \(L\)。
3.  计算金属丝的直径 \(D\) 及其不确定度，分析误差来源。

## 数据处理

### 一、牛顿环测曲率半径

#### 1. 测量方法

采用**逐差法结合最小二乘法**处理数据。测量第20～30环暗环的弦长（代替直径），记录左右两端的读数 \(a_i\) 和 \(b_i\)，则第 \(i\) 环的弦长为：

\[ D_i = |a_i - b_i| \]

> **重点**：实验中用测量弦长代替测量直径，结果完全一致。这是因为弦长 \(D'^2 = D^2 - 4h^2\)（\(h\) 为弦到中心的距离），在逐差法中 \(D'^2_{k+m} - D'^2_k = D^2_{k+m} - D^2_k\)，常数项 \(4h^2\) 被消去。

#### 2. 公式推导

由暗环条件 \(r_k^2 = k\lambda R\)，直径 \(D_k = 2r_k\)，有：

\[ D_k^2 = 4k\lambda R \]

取级次差 \(m\)，得：

\[ D_{k+m}^2 - D_k^2 = 4m\lambda R \]

令 \(k_0 = 20\)，\(y_m = \frac{D_{k_0+m}^2 - D_{k_0}^2}{4\lambda}\)，\(x = m\)，则 \(y = Rx + a_0\)。

用最小二乘法拟合直线，斜率即为曲率半径 \(R\)：

\[
R = \frac{n \sum x_i y_i - \sum x_i \sum y_i}{n \sum x_i^2 - \left(\sum x_i\right)^2}
\]

#### 3. 误差分析

-   **系统误差**：
    -   透镜与平板非理想点接触（存在接触面或灰尘），导致中心不是理想暗点。
    -   读数显微镜的空程差（螺距误差）。
    -   钠光并非严格单色光（钠双线 \(\lambda_1 = 589.0\,\text{nm}\)，\(\lambda_2 = 589.6\,\text{nm}\)）。
    -   透镜球面非理想球面。

-   **随机误差**：
    -   对叉丝瞄准条纹中心位置的人眼判读误差。
    -   测量时鼓轮单方向移动不均匀。

-   **不确定度计算**：由最小二乘法斜率不确定度公式：

\[
u(R) = \sqrt{\frac{\sum (y_i - R x_i - a_0)^2}{(n-2) \sum (x_i - \bar{x})^2}}
\]

### 二、劈尖干涉测金属细丝直径

#### 1. 测量方法

测量 \(N\) 条（如50或100条）暗纹之间的总间距 \(s\)，则相邻条纹间距 \(l = s / N\)。再测量金属丝到棱边的距离 \(L\)。

#### 2. 公式推导

由式 (18-12)，金属丝直径为：

\[
D = \frac{L \lambda}{2l} = \frac{L \lambda N}{2s}
\]

#### 3. 误差分析

-   条纹间距 \(l\) 的测量误差（可通过多组测量取平均值减小）。
-   金属丝到棱边距离 \(L\) 的测量误差。
-   劈尖两玻璃板可能不完全平整，导致条纹不均匀。

---

## 注意事项

1.  **读数显微镜空程差**：测量过程中鼓轮只能单方向移动，严禁中途反向，否则会引入空程误差。

> **易错**：从35环开始向20环测量时，必须始终沿同一方向旋转鼓轮，若过头也不能回退，只能重新从35环开始。

2.  **调焦顺序**：先调节目镜焦距以看清叉丝，再调节物镜调焦旋钮以看清成像，消除视差。
3.  **被测物放置**：被测物表面应与显微镜光轴垂直。
4.  **钠光灯预热**：钠光灯需要预热一段时间才能稳定发光，使用过程中不要频繁开关。
5.  **保护光学面**：牛顿环仪和劈尖装置的光学表面不可用手直接触摸，避免划伤和污染。
6.  **白光观察**：用白光代替钠光时，白光相干长度短，只能看到少数几圈彩色环纹。

## 思考题

### 预习思考题

1.  使用读数显微镜要注意哪些问题？如何使用读数显微镜来测量牛顿环的直径（弦长）？

    ??? note "参考答案"
        注意事项：（1）测量过程中鼓轮只能单方向移动，避免空程差；（2）先调节目镜以看清叉丝，再调节物镜以看清成像并消除视差；（3）被测物表面应与显微镜光轴垂直。测量方法：转动鼓轮使十字叉丝从中心向外侧移动到第35环外，再单方向移回，依次测出第30环到第20环左右两端的读数 \(a_i\) 和 \(b_i\)，弦长 \(D_i = |a_i - b_i|\)。

2.  牛顿环干涉条纹形成在哪一个面上（即定域在何处）？

    ??? note "参考答案"
        牛顿环干涉条纹定域在空气层附近（即空气薄膜的上表面附近）。因为入射光在空气层上下表面反射后，两束反射光在空气层上表面附近相遇叠加产生干涉，所以条纹定域在空气层上表面处。

3.  实验中用测量弦长代替测量直径，结果完全一样，请证明。

    ??? note "参考答案"
        设弦到中心的距离为 \(h\)，弦长 \(D'\) 与直径 \(D\) 的关系为 \(D'^2 = D^2 - 4h^2\)。在逐差法中计算 \(D_{k+m}^2 - D_k^2\) 时，由于所有弦到中心的距离 \(h\) 相同（叉丝水平移动），有 \(D'^{\,2}_{k+m} - D'^{\,2}_k = (D^2_{k+m} - 4h^2) - (D^2_k - 4h^2) = D^2_{k+m} - D^2_k\)，常数项 \(4h^2\) 被消去，因此用弦长代替直径结果完全一样。

4.  利用劈尖干涉如何测量头发丝的直径？

    ??? note "参考答案"
        将两片平板玻璃叠合，在一端垫入头发丝形成空气劈尖。用单色光垂直照射，观察到与棱边平行的明暗相间直条纹。测量 \(N\) 条暗纹的总间距 \(s\)（则相邻条纹间距 \(l = s/N\)），再测量头发丝到棱边的距离 \(L\)，由公式 \(D = L\lambda/(2l) = L\lambda N/(2s)\) 即可算出头发丝直径。

5.  劈尖干涉的条纹定域在何处？试画出光路图说明之。

    ??? note "参考答案"
        劈尖干涉条纹定域在空气劈尖薄膜的上表面附近。入射光在劈尖上下表面反射后，两束反射光在劈尖上表面附近相遇叠加，由于劈尖各处厚度不同，形成与棱边平行的等厚干涉条纹。光路：单色光垂直入射到上玻璃板表面，一部分从上表面反射，另一部分折射进入空气层后从下表面反射，再折射出上表面，两束反射光在上表面附近相遇产生干涉。

### 思考题

1.  在牛顿环实验中，如果平板玻璃上某处有微小凸起，则凸起处空气薄膜的厚度减小，导致等厚干涉条纹发生变化。试问这时的牛顿环将局部内凹，还是外凸？为什么？

    ??? note "参考答案"
        牛顿环将局部内凹。等厚干涉条纹是等厚度的轨迹。凸起处空气膜厚度减小，同一厚度对应的干涉环向厚度更大的外侧"借"位置，因此条纹向中心方向偏移（内凹）。反之，若平板有凹陷，则该处空气膜厚度增大，条纹向外凸。

2.  从牛顿环透射过来的光是否可形成干涉条纹？若能，其干涉条纹与反射光的条纹有什么不同？请画出透射式干涉环光路图并解释中心圆环属于亮环还是暗环。

    ??? note "参考答案"
        透射光也能形成干涉条纹。反射光中从空气层下表面反射的光有半波损失，而透射光中两束透射光的光程差不含半波损失项（或两束透射光的半波损失互相抵消），因此透射条纹与反射条纹互补——反射暗环处透射为亮环，反射亮环处透射为暗环。反射光中心为暗斑（有半波损失），透射光中心为亮斑。

3.  利用劈尖空气薄膜形成的干涉条纹可以检查工件表面的加工质量。实验装置如图18-5a所示，在工件表面上放置一块平面度高的平板玻璃，使其间形成空气劈尖。用单色光垂直照射玻璃表面，用显微镜观察干涉条纹。假设观察到的条纹如图18-5b所示，试根据条纹弯曲的方向，说明工件表面上纹路是凹的还是凸的？

    !!! note "原理示意图：检查工件表面加工质量"
        页4，检查工件表面加工质量的劈尖干涉装置及条纹弯曲示意图

    ??? note "参考答案"
        等厚干涉条纹是等厚线。若条纹向棱边方向（薄处）弯曲，说明该处厚度比正常偏大，即工件表面凹下；若条纹向远离棱边方向（厚处）弯曲，说明该处厚度比正常偏小，即工件表面凸起。判断口诀："条纹弯向薄处，工件表面凹下；条纹弯向厚处，工件表面凸起。"
